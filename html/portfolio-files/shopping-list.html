About

This is a complete app that allows you to dinamically create a list of items, intended for when you go shopping. You can add items with their names, their prices, and the number of units. Added items can be modified and deleted. You can also choose multiple currencies to display and print the list when it is finished. In short, it is a simple but functional app, and totally responsive.

This was created with the idea of practicing real-time manipulation of the nodes in the DOM, and adding events listeners to those nodes. When you check the code on github, place make sure that you check also the last date when this app was updated, because there might be changes.
