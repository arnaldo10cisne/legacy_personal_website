THE MARVEL GUIDEBOOK

About this project

Tech

HTML, CSS, Javascript

Year

2021

Language

English

This website was created to offer easy access to the MARVEL API, and test fetching data in the ".json" format. I also wanted to do something different regarding the styles. So the approach was a colorful and informal presentation, reminiscent of my idea of a comic book.

Finally, why choosing the MARVEL API over the other hundreds of free APIs available for testing? The short answer is... I like the Marvel Universe, and I grew up with these characters. I believe everyone can run the extra mile, and deliver a better product if there is a little bit of passion involved.
